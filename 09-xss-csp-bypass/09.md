# 09-xss-csp-bypass

## 核心思路

页面是一个公告评论系统，评论内容 `content` 支持部分 HTML。

响应头里的 CSP：

```http
Content-Security-Policy: default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'; img-src * data:;
```

问题点：

1. `script-src` 有 `'unsafe-inline'`，所以内联事件可以执行。
2. 过滤器只过滤了带引号事件处理器，比如 `onerror="..."`。
3. 无引号事件处理器可以绕过，比如：

```html
<img src="x" onerror="alert(document.cookie)" />
```

另外，响应头直接下发了：

```http
Set-Cookie: ADMIN_TOKEN=TOGOGO-flag%7B46ebf8a1-20c7-424c-b437-c78a93eb1f72%7D; path=/
```
